Manual Conexión HTTP con JAVA

Manual para realizar una conexión mediante un objeto de acceso a datos (DAO) a cada uno de los ENDPOINT de un servicio RESTful para cada operación CRUD(crear, leer, actualizar y eliminar):

La siguiente tabla la operación CRUD que realiza cada método HTTP y si cuenta con cuerpo de solicitud (Request Body) o cuerpo de respuesta (Response Body) al realizar una conexión HTTP a traves de una URL:

METODO HTTP OPERACIÓN CRUD Request Body Response Body
POST CREATE TRUE FALSE
GET READ FALSE TRUE
PUT UPDATE TRUE FALSE
DELETE DELETE FALSE FALSE
